Frequently Asked Questions

In order to find registered shipment, please type in the box the number (for example: 0015900773312345678, PX0000000013, RR123456789PL, CP123456789PL, VV123456789PL, EE123456789PL) listed on the postal receipt, without any spaces or parentheses, and press Enter or tap on Enter Button.

If the following message diplayed, the number is incorrect or the system does not list a shipment with the specified number.

If the number is correct, the shipping information and event log for the parcel will be displayed. If the shipping information does not conform to the information on the postal receipt, please check if you have entered the number correctly.

In the case of registered letters, only dispatch and delivery date may be displayed, without tracking information.

The system currently provides information about the following types of shipments:

  • Domestic
    • Postal Parcel
    • Pocztex
    • Cash on Delivery Package
    • Declared Value Packages
    • Business Packages
    • Dimensional Weight Package
  • International
    • EMS
    • Postal Parcel
